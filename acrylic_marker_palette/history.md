# アクリルマーカー色見本（64色セット）のモデリング履歴

## 2026/04/28: 新規作成

### 設計要件と経緯
- **目的**: Arrtxアクリルマーカー64色セット（画像の色見本）を、実際にマーカーで色を塗って確認できる物理的なカラーパレットとして作成。
- **構成**:
  - 120mm x 120mm x 2mmの板を4枚。
  - 各板には16色分（4列×4行）のマスを配置。
  - 1マスにつき「塗るための四角い凹み（24x14mm、深さ0.4mm）」と「浮き出た色名のテキスト（高さ0.6mm）」を設ける。
- **パラメータの意図**:
  - **凹み**: 後からアクリルマーカーで塗る際に、はみ出しを防ぎきれいに仕上がるようにするため。
  - **浮き出しテキスト**: 印刷時にAMS機能等を用いて文字だけを簡単に黒などの別色に塗り分けられるようにするため。

### 推奨フィラメントと印刷設定（Bambu Lab P2S 等）
- **フィラメント**:
  - ベース: PLA Basic (White) など、インクの発色が分かりやすい白色系。
  - 文字: PLA Basic (Black) などのコントラストがはっきりする色。
- **スライサー設定**:
  - **0.16mm Optimal** 等、積層ピッチを少し細かくすることで文字のディテールがきれいに印刷されます。
  - **着色方法**:
    - 「塗りつぶし（Fill）」ツールの「角度（Angle）」設定を利用すると、0.6mm浮き出た文字部分だけを一括でクリックして別フィラメントに指定可能です。
    - もしくは、Z軸の高さモディファイアを使用してレイヤーごとの色変更を行うことでも対応可能です。
