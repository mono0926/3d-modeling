# アクリルマーカー色見本（64色セット）のモデリング履歴

## 2026/04/28: 新規作成

### 設計要件と経緯
- **目的**: Arrtxアクリルマーカー64色セット（画像の色見本）を、実際にマーカーで色を塗って確認できる物理的なカラーパレットとして作成。
- **構成**:
  - 120mm x 120mm x 2mmの板を4枚。
  - 各板には16色分（4列×4行）のマスを配置。
  - 1マスにつき「塗るための四角い凹み（24x14mm、深さ0.4mm）」と「浮き出た色名のテキスト（高さ0.6mm）」を設ける。
- **パラメータの意図**:
  - **凹み**: 後からアクリルマーカーで塗る際に、はみ出しを防ぎきれいに仕上がるようにするため。
  - **浮き出しテキスト**: 印刷時にAMS機能等を用いて文字だけを簡単に黒などの別色に塗り分けられるようにするため。

### 推奨フィラメントと印刷設定（Bambu Lab P2S 等）
- **フィラメント**:
  - ベース: PLA Basic (White) など、インクの発色が分かりやすい白色系。
  - 文字: PLA Basic (Black) などのコントラストがはっきりする色。
- **スライサー設定**:
  - **ウォールジェネレーター**: `Arachne` 推奨。細かい文字の線幅を動的に変えて綺麗に造形してくれます。
  - **積層ピッチ**: `0.20mm Standard` で十分綺麗に出力可能です。（文字はZ方向に垂直に押し出されるため、ピッチを細かくしても形状自体のXY解像度には直結しません）
  - **スピード**: 文字のエッジをシャープにするため、`Top surface` を 50 mm/s、`Outer wall` を 100 mm/s 程度に落とすことを強く推奨します。
  - **アイロニング**: 「最上層のみ (Topmost surface only)」をオンにすると文字の天面が滑らかに仕上がります。
  - **パージ量**: 今回は「白ベース」の上に「黒文字」が乗るため、フィラメントの切り替えは1回（白→黒）しか発生しません。黒は白に対して圧倒的に色が勝ちやすいため、マルチプライヤはデフォルト（1.00）のままか、0.5 等に下げても問題ありません。

## 2026/04/28: 最適化と仕様変更のやり取り
- **アセンブリ出力への変更**:
  - 当初は1つのボディとしてSTEP出力し、Bambu Studioの「塗りつぶし」ツールで色分けする想定でしたが、ユーザーの手間を省くため、ベースとテキストを `cq.Assembly()` でまとめて色情報を付加して出力する方式に変更しました。これにより、Bambu Studio読み込み時に自動でマルチパーツとして認識され、黒色が割り当てられます。
- **ベース板の厚み変更 (2.0mm -> 1.2mm)**:
  - フィラメント使用量と印刷時間節約のため、板の厚みを 1.2mm に変更しました。
  - 凹みの深さが 0.4mm のため、凹み底の厚みは 0.8mm となります。これは 0.2mm ピッチで4レイヤー分に相当し、底に穴が開くこともなく十分な強度を保てるため、実用上全く問題ありません。
