# Skadis Base Hook

## 履歴とプロンプト経緯

### 2026-03-02

- **ユーザー要望**: IKEA Skadis用フックの返しの長さを6.4mmではなく、ベストな長さでモジュールとして使い回せるCadQueryコードが欲しい。
- **対応内容**:
  - 返しの長さを保持力と着脱のしやすさのベストバランスである `5.0mm` に設定。
  - 首の長さを `5.4mm` とし、ボード（5.0mm厚）との干渉を防ぐ設計を採用。
  - スコーディスの穴の底面の半円弧形状とプリント時の長方形形状（幅4.8mm）との関係を利用し、フックモデルが穴の中で約1.8mm浮き上がる特性を活用。これにより、内側コーナーに強固なフィレット（R1.5）を入れてもボードと干渉しない、非常に実用的かつ堅牢な形状を実現した。
  - `get_base_hook()` 関数として定義し、他の形状（J字フックなど）と `.union()` や `.add()` で簡単に合成して使い回せるようにした。
  - サンプルのJ字フックも同時に生成・エクスポートするコードとして提供。
