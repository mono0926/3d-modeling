# 設計履歴 - Studio Display 収納トレイ

## V4.1 (2026/03/27) - 究極の微調整
- **主な変更:**
  - 壁厚さを 2mm に変更。
  - 前面傾斜角を 60度 に変更。
  - 棚の段差（厚み）を 6mm に設定。
  - 背面支柱逃げ R5 (半径) を維持。
  - 前面底部 R10 (半径) を維持。
  - 外寸: 152mm(W) x 105mm(H) x 175.71mm(D)

## V4 (2026/03/27) - 確定寸法・安定化
- **主な変更:**
  - ユーザー提供の確定寸法 (152x105x175.71) を反映。
  - 支柱 R5 逃げとベース R10 フィレットを実装。
  - 壁厚 2.4mm、前面傾斜 65度。
スタンドの支柱付け根のR5を逃がすための面取りを背面に配置し、究極のフィット感を実現。
- **モデリング (V3)**: 前回の「中身の詰まった箱」問題を解消。くり抜き用の内部ソリッドを前面方向に大きく突き出させることで、前面を確実に「開口」させ、トレイとして機能するように修正。
- **モデリング (V2)**: `shell` 関数の不安定さを避けるため、YZ断面プロファイルを押し出した「外形」から「内形」を差し引くブーリアン演算手法に全面的に書き換え。背面を確実に垂直（90度）にし、指定の奥行き170mmを確保。棚板も内部空間でトリミングして配置。
- **ユーザー要望**: Apple Studio Displayの「傾きと高さを調整できるスタンド」の空きスペースを埋めるトレイを作りたい。MakerWorldの既存モデルは「傾きを調整できるスタンド」用で背面が斜めになっているため隙間ができる。奥の角度を直角（90度）にし、奥行きを約170mmに伸ばしたいという要望。
